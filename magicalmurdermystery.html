Description

Star Apprentice: Magical Murder Mystery is a danmaku-detective game that combines point-and-click style deduction with high octane bullet hell gameplay. Find evidence, interrogate suspects, and refute contradictions to solve the case!

Screenshots

History

Star Apprentice: Magical Murder Mystery was first created as a student project for the University of Alberta Games Certificate intro course, where it won the CMPUT250 Game of the Year award in 2021. The game was soon released on Steam.

In late 2022, the now lead developer Jamie Powers started porting the game to a new engine that was being made in preparation for a sequel. This version 2.0 update was then released on Feb 18, 2023.
